6) [GIT](/docs/git.md) - мелочи, которые постоянно забываешь

8) [WireGuard](/docs/WireGuard.md) - Настройте вашему ребенку VPN, а то чо он у вас как дебил?! 
9) [SSH](/docs/ssh.md) - ssh доступ по ключам
